راهنما

  1. مرحلهٔ ۱: دو کاربر انتخاب کن — محتوای مشترکشان نمایش داده می‌شود.
  2. مرحلهٔ ۲: دو محتوا انتخاب کن — کاربران مشترک نمایش داده می‌شوند.
  3. مرحلهٔ ۳: دو جفت انتخاب کن (ابتدا یک جفت، سپس دوم) و سپس دکمهٔ «مقایسه» را بزن.
  4. برای پاک کردن انتخاب‌ها، روی فضای خالی کلیک کن یا از دکمهٔ «بازنشانی» استفاده کن.
